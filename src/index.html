Guide complet Classiflyer

📁 Mes Classeurs

Votre espace de travail principal pour organiser, consulter et gérer tous vos documents de manière structurée et intuitive.

🎯 Vue d'ensemble

  • Organisation complète : Créez et gérez vos classeurs de documents avec une structure illimitée
  • Consultation avancée : Visualisez tous types de fichiers directement dans l'application
  • Navigation fluide : Parcourez vos documents avec des outils de navigation intuitifs
  • Recherche intelligente : Trouvez rapidement vos classeurs parmi tous vos documents
  • Gestion flexible : Modifiez, archivez ou supprimez selon vos besoins

➕ Création de classeurs

Deux méthodes pour créer un nouveau classeur :

  • Classeur vide : Créez un classeur vierge que vous remplirez progressivement
  • Depuis un dossier existant : Transformez un dossier de votre ordinateur en classeur Classiflyer
1. Cliquez sur "Nouveau classeur"
2. Choisissez "Classeur vide" ou "Depuis un dossier"
3. Si depuis un dossier : sélectionnez le dossier sur votre ordinateur
4. Donnez un nom à votre classeur
5. Validez la création

🔍 Recherche de classeurs

Trouvez rapidement vos classeurs parmi tous vos documents :

  • Recherche par nom : Tapez le nom ou une partie du nom du classeur
  • Filtrage instantané : Les résultats s'affichent en temps réel
  • Recherche globale : Recherchez dans tous vos classeurs simultanément

🎮 Contrôles de navigation

Naviguez facilement dans vos documents :

  • Flèches de navigation : Utilisez les flèches ← → pour passer d'un onglet à l'autre
  • Navigation dans les PDF : Parcourez les pages d'un document PDF multi-pages
  • Zoom sur les images : Agrandissez ou réduisez les images pour une meilleure lecture
  • Défilement : Faites défiler les documents longs verticalement
  • Glisser-déposer : Déplacez facilement les fichiers entre les dossiers

📁 Gestion des dossiers et fichiers

Organisez vos documents de manière illimitée :

  • Dossiers imbriqués : Créez des sous-dossiers à l'infini
  • Icône de création : Passez la souris sur un dossier pour voir l'icône "+" de création de sous-dossier
  • Upload multiple : Téléchargez plusieurs fichiers simultanément
  • Renommage : Renommez fichiers et dossiers en un clic
  • Suppression sélective : Supprimez des éléments individuellement

Éléments supprimés

📁 Organisation des archives

  • Dossiers d'archives : Créez des catégories (Ex: "2024", "Projets terminés")
  • Structure préservée : Tous les sous-dossiers et fichiers restent intacts
  • Chemins mis à jour : La base de données est automatiquement synchronisée

🗑️ Corbeille

Système de sécurité pour récupérer les éléments supprimés par erreur.

🛡️ Sécurité des données

  • Suppression temporaire : Les éléments ne sont pas définitivement perdus
  • Anciens chemins sauvegardés : Possibilité de restauration complète
  • Récursivité préservée : Toute la structure des dossiers est conservée
  • Double protection : Depuis "Mes Classeurs" ou "Archives"

🔄 Processus de suppression

  1. Déplacement physique : Le dossier est déplacé vers le dossier corbeille
  2. Sauvegarde des chemins : old_sys_path est enregistré pour chaque fichier/dossier
  3. Mise à jour de la DB : Les nouveaux chemins sont mis à jour
  4. Marquage temporel : Date de suppression enregistrée

♻️ Restauration

  • Restauration complète : Remet le classeur exactement comme avant
  • Chemins restaurés : Utilise les old_sys_path sauvegardés
  • Structure préservée : Tous les sous-dossiers et fichiers retrouvent leur place
  • Nettoyage automatique : Les anciens chemins temporaires sont supprimés

⚠️ Suppression définitive

La suppression définitive efface complètement les fichiers du disque. Cette action est irréversible.

⚙️ Paramètres

Configuration centrale de Classiflyer.

📂 RootPath

Le RootPath est le dossier principal où Classiflyer stocke tous ses fichiers :

  • classeurs/ : Vos classeurs actifs
  • archives/ : Les projets archivés
  • corbeille/ : Les éléments supprimés
  • db.json : Base de données de l'application

💾 Structure recommandée

~/.classiflyer/config/          (Linux/Mac)
C:\Users\[user]\AppData\Roaming\Classiflyer\config\  (Windows)
├── db.json
├── classeurs/
│   ├── Projet 2025/
│   └── Factures/
├── archives/
│   ├── 2024/
│   └── Projets terminés/
└── corbeille/

🔧 Configuration

  • Chemin personnalisé : Choisissez où stocker vos données
  • Portabilité : Déplacez facilement votre configuration
  • Sauvegarde : Sauvegardez le dossier entier pour une sauvegarde complète
